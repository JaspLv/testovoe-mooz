## Решение:

Список страниц:

- "/" - Главная - каталог фильмов
- "/login"- страница входа

1. Страница входа:

- Логин (USERNAME: любой)

2. Страница каталога:

- Поиск
- Юзернейм
- Каталог фильмов и пагинация

Для запуска необходимо настроить `.env` файл на основе `.env.example`

Запуск:

`npm run dev`
